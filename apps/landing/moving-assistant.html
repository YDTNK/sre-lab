AI Moving Assistant

AI引越し診断

家具、衣類、家電、本・小物、引越し日、補足メモを入力すると、 引越し準備のサマリー、必要な梱包資材、チェックリスト、注意点を整理します。

OpenAI APIをバックエンド経由で利用し、fallback response、rate limiting、 利用量・推定コスト記録を備えたSRE Labの1つ目のサービスです。

引越し情報を入力

分かる範囲で入力してください。すべての項目を埋める必要はありません。 家具・衣類・家電などの情報が多いほど、より具体的なチェックリストを作成できます。







診断結果

入力後、「チェックリストを作成する」を押してください。

このサービスで実装している運用要素

  • Cloudflare Workers API
  • OpenAI APIのバックエンド連携
  • API keyのSecret管理
  • Rate limiting
  • fallback response
  • 利用量・推定コスト記録
  • Grafana Synthetic Monitoring
  • Grafana Alerting
