AWS Cost Simulator

AWS Cost Simulator is a planned educational tool for estimating simple AWS monthly costs.

The MVP will start with deterministic calculations for EC2, EBS, S3, and data transfer placeholders. It will not require a paid AI API in the first version.

Back to SRE Lab

Planned MVP Scope

Inputs

  • Region
  • EC2 instance type
  • EC2 instance count
  • EC2 hours per month
  • EBS storage GB
  • S3 storage GB
  • Data transfer GB

Outputs

  • Estimated monthly USD cost
  • Estimated monthly JPY cost
  • Resource cost breakdown
  • Assumptions
  • Educational disclaimer
