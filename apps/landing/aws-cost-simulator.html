AWS Cost Simulator / AWS料金試算

AWS Cost Simulatorは、AWS月額料金を教育用に概算する小さなサービスです。

EC2、EBS、S3、Data transferを固定単価ベースで概算します。 このサービスでは有料AI APIを使用しません。
