SRE Lab

小さなAI / SREサービス運用ラボ

SRE Lab is a small multi-service operations lab for building, monitoring, and improving AI/SRE-related services.

This project focuses on public deployment, CI/CD, monitoring, alerting, runbooks, incident logs, API safety, AI cost control, and continuous operations.
