SRE Lab

小さなAI / SREサービスを運用するポートフォリオ

SRE Labは、複数の小さなAI / SRE系サービスを開発・公開し、CI/CD、監視、Alert、Runbook、運用記録まで含めて継続的に改善していくポートフォリオです。

単なるWebアプリ制作ではなく、公開後の運用・監視・改善まで含めて、SRE / Platform Engineeringの実践力を示すことを目的としています。

公開中のサービス

公開中 / AI API連携

AI Moving Assistant / AI引越し診断

家具、衣類、家電、本・小物、引越し日などを入力すると、引越し準備のサマリー、必要資材、チェックリスト、注意点を整理します。

  • OpenAI APIをバックエンド経由で利用
  • fallback response
  • Rate limiting
  • 利用量・推定コスト記録
AI引越し診断を開く

公開中 / 監視・Alert設定済み

AWS Cost Simulator / AWS料金試算

EC2、EBS、S3、Data transferの入力値をもとに、AWS月額料金の概算を固定単価ベースで計算します。

  • Workers APIによる料金計算
  • 入力値の範囲検証
  • 有料AI API不使用
  • Synthetic Monitoring / Alerting
AWS料金試算を開く

運用で重視していること

SRE Labでは、機能を作るだけでなく、公開後に安全に運用し続けるための仕組みを整えています。

  • Cloudflare Pages / Workersによる公開
  • GitHub ActionsによるCI/CD
  • Grafana Synthetic Monitoringによる外形監視
  • Grafana Alertingによるアラート通知
  • RunbookとOperational Recordsによる運用記録
  • API安全化、Rate limiting、Cost control
